## Ory SDK & OpenAPI Maintainer

<About />

In this role you will use your passion for open source to maintain Ory SDKs,
with a focus on the OpenAPI Generator, as it provides SDKs that are integral to
Ory’s identity management infrastructure. Since we provide SDKs for all common
programming languages, being a polyglot is essential for this role.

### Overview

- **Position Type:** Full Time
- **Location**: Remote (Timezone CET ± 3H) / Munich, Germany
- **Components**: Ory Open Source, GitHub

### What You'll Do

- Design, build and maintain production code for Ory SDKs, with a strong initial
  focus on the OpenAPI generator project.
- Collaborate with the Ory Open Source community to build and expand the Ory
  Open Source ecosystem.
- Be an enthusiastic ambassador for Ory products in the open source community.
- Provide technical leadership and solve end-to-end problems (e.g., software
  design, efficient implementation, and product offering).
- Write documentation in code.

### We’re looking for someone who has

- A Bachelor’s degree in Computer Science or related technical field or
  equivalent practical experience.
- A high level of expertise with Java, the foundation of the OpenAPI Generator.
- Experience with other programming languages including Go, Node, PHP, Ruby,
  Python, JavaScript.
- The ability to quickly learn and adapt new languages (e.g. Rust).
- Enthusiasm for OSS and desire to maintain a large project like OpenAPI
  Generator.
- The ability to write excellent developer documentation.
- Interest in the latest web standards and technologies.
